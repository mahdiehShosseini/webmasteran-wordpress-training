مقدمه

جاوا اسکریپت یک زبان اسکریپتی چند سکویی و شی گراست، سبک و کوچک است.در هر محیطی (مثلا مرورگر) که قرار گیرد میتواند به اشیای آن محیط متصل شود و قابلیت کنترل آن ها را بدست گیرد.

  • دانش خوبی از زبان نشانه گذاری HTML
  • تجربه برنامه نویسی
جاوا و جاوا اسکریپت

جاوا و جاوا اسکریپت شباهت ها و تفاوت هایی دارند، زبان جاوا اسکریپت تا حدودی مشابه زبان جاواست به طوری که سینتکس و ساختار کنترلی و سیاست های نام گذاری آن بسیار به جاوا شباهت دارد اما تفاوت هایی دارد از جمله اینکه همانند جاوا از تعریف نوع متغییر به صورت استاتیک (یعنی مشخص کردن نوع متغییر در زمان کامپایل) پشتیبانی نمی کند، همین شباهت ها یکی دلایل اصلی تغییر نام آن از LiveScript به JavaScript بوده است.

جاوا اسکریپت بر خلاف سیستم زمان کامپایل جاوا برای کلاس ها، از سیستم زمان اجرا (Runtime) خاصی استفاده میکند که در آن بر اساس تعداد محدودی از انواع داده ها از جمله رشته ها ، مقادیر عددی و بولین کار میکند. جاوا اسکریپت از مدل Prototype-Based Object Model استفاده میکند در حالیکه جاوا از مدل Class-Based Object Model استفاده می کند، مدل دوم رایج تر است ولی مدل اول برای ما ارث بری پویا را برای ما به ارمغان میاورد به این معنی که ارث بری برای هر شی می تواند متفاوت باشد.

علاوه بر اینها جاوا اسکریپت زبان به مراتب ساده تری از جاواست چرا که برنامه نویسی دیگر نیازی ندارد به بسیاری از مباحث از جمله کپسوله سازی و یا تعریف نوع متغییر توجه داشته باشد چرا که جاوا اسکریپت سطوح دسترسی همچون public - protected -private - package level را ندارد.

سلام دنیا!!

نمونه تابعی که Hello World را چاپ کند و فراخوانی آن به شکل زیر است:

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");

شما می توانید آن را داخل فایل متنی کپی کنید و با مرورگر خود آن را اجرا کنید!

این دکمه همین کار را می کند!
